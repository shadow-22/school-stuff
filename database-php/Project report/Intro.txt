Για την εργασία χρησιμοποιήθηκε η βάση δεδομένων MySQL και η γλώσσα προγραμματισμού PHP. Πρόκειται για έναν από τους περισσότερο διαδεδομένους συνδοιασμούς λογισμικού και επομένως υπάρχει πολυ υποστηρικτικό υλικό στο διαδικτύο για την βασική εκμάθηση αλλά και για περεταίρω εξάσκηση μέσα από forums και διάφορες άλλες κοινότητες. Σημαντικό κρίνεται επίσης το γεγονός ότι και τα δύο πακέτα διατίθενται δωρεάν. Επίσης, τα προγράμματα PHP εκτελούνται στον server σε αντίθεση με την Javascript που εκτελείται στον client και επομένως ενδείκτυται για χρησιμοποίηση βάσεων δεδομένων που υπό κανονικές συνθήκες δεν θα υπήρχαν στον υπολογιστή του πελάτη. Τέλος, μέσω του Xampp που συνδοιάζει μαζί με τα παραπάνω το PHPmyAdmin μπορούμε να δημιουργήσουμε απλές βάσεις δεδομένων εύκολα και γρήγορα χωρίς υπερβολικά βαριά προράμματα όπως οι λύσεις της Microsoft. 

Όσο αναφορά τα προβλήματα που αντιμετώπισα σε πρώτη φάση ήταν η σωστή επιλογή λογισμικού αλλά κυρίως η δημιουργία των triggers.
